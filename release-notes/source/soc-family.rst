Soc Family
============================

SOC_FAMILY is a variable used to cluster the similar machines into families and allow a inheritance for similar hardware configuration.

The current SOC_FAMILY tree can be see in the next graph:

.. code-block:: none

  SOC_FAMILY
  ├── mx3
  │   ├── mx31
  │   └── mx35
  ├── mx5
  │   ├── mx51
  │   └── mx53
  ├── mx6
  │   ├── mx6dl
  │   ├── mx6q
  │   ├── mx6s
  │   └── mx6sl
  ├── mxs
